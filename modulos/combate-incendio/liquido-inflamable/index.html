Técnicas de Aplicación

Método Directo

  • Aplicar directamente sobre líquido
  • Para derrames pequeños
  • Puede agitar el combustible
  • Riesgo de salpicadura

Método Indirecto (Roll-On)

  • Espuma rebota en superficie sólida
  • Se desliza sobre el líquido
  • Menos agitación
  • Más seguro para grandes derrames

Método Rain-Down

  • Espuma cae como lluvia desde arriba
  • Cobertura uniforme
  • Para tanques abiertos
  • Requiere altura adecuada

Método Bank-Down

  • Espuma contra pared o dique
  • Se extiende naturalmente
  • Para derrames en diques
  • Menos turbulencia

Cálculo de Concentración

Tipo Espuma Concentración Ejemplo Cálculo
AFFF 3% 3 litros por cada 100L agua 1000L agua = 30L espumante
AFFF 6% 6 litros por cada 100L agua 1000L agua = 60L espumante
Clase A 0.5% 0.5 litros por cada 100L agua 1000L agua = 5L espumante
Clase A 1% 1 litro por cada 100L agua 1000L agua = 10L espumante

Errores Comunes

Errores de Aplicación

  • Aplicar espuma con mucha presión
  • Romper la capa de espuma existente
  • Usar concentración incorrecta
  • No cubrir toda la superficie
  • Dejar zonas sin cobertura

Errores Tácticos

  • Usar agua sola en Clase B
  • No controlar derrames primero
  • Atacar contra el viento
  • No preparar líneas de respaldo
  • Ignorar vapores inflamables
NUNCA usar agua sola en líquidos Clase B - Esparce el fuego. SIEMPRE usar espuma o agentes químicos

Seguridad en Líquidos Inflamables

Evaluación Previa

  • Identificar el líquido: Placas, etiquetas, documentos
  • Punto de inflamación: ¿Puede encenderse a temp ambiente?
  • Cantidad derramada: Estimar volumen
  • Fuentes de ignición: Eliminar chispas, llamas
  • Vías de escape: Siempre planificadas
  • Dirección del viento: Ataque desde barlovento
REGLA DE ORO: Identificar el líquido ANTES de atacar. Usar espuma apropiada. Cubrir TODA la superficie
